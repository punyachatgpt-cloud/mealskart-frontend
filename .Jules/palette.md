# Palette's UX Journal

## 2025-05-14 - Standardized Haptic Feedback for Onboarding
**Learning:** Onboarding flows often lack the same level of interactive feedback as the main application, leading to a disjointed user experience. Standardizing tactile feedback (40ms vibration) across both the main app and the onboarding sequence reinforces the app's 'premium' and 'responsive' feel.
**Action:** Always verify that auxiliary pages (onboarding, profile, settings) implement the same haptic feedback patterns as the primary interface.
