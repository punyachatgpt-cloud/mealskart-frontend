# Palette's UX Journal

## 2025-05-15 - [Keyboard Shortcuts & A11y]
**Learning:** Combining keyboard shortcuts with clear visual hints (like `title` attributes) improves power-user efficiency without sacrificing discoverability for casual users. Additionally, dynamic `aria-label` updates for state-dependent buttons (like user avatars) provide a more personalized and accessible experience.
**Action:** Always pair new keyboard shortcuts with a discoverable hint (tooltip or label) and ensure all primary navigation buttons have descriptive, state-aware ARIA labels.
